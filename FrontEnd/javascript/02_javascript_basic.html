자바스크립트 자료형

자바스크립트는 변수 선언 시 자료형을 작성하지 않음 무조건 var

1. 문자열 : "", ''로 묶여있는 리터럴(자바와 다르게 '', "" 구분안하고 모두 문자열로 처리함)

string이라는 내장 객체로 기본적인 함수(메소드)들이 존재

2. 숫자 : 정수형 숫자와 부동소수점 숫자 처리하는 자료형

Math라는 내장객체를 이용하여 기본적인 메소드 사용이 가능

3. 기타자료형 : 논리형(true/false), 객체(object), undefined, 함수(function)

자바스크립트 연산자

1. 대부분 연산자는 자바 연산자와 동일

2. ==, != : 자료형과 데이터가 모두 일치하는지 확인하는 연산자

3. typeof() : 해당하는 값이 어떤 자료형인지 확인하는 연산자

자바스크립트 형변환

문자열 -> 숫자 형변환 : Number(문자) / 숫자 -> 문자열 형변환 : String(숫자)

자바스크립트 변수

지역변수 & 전역변수

자바스크립트 제어문

1. 조건문(if, if-else, if-else if, switch, 짧은 조건문(||, &&) : 짧은 조건문을 제외하면 자바와 동일

2. 반복문(for, while) : 자바와 동일

3. 분기문(continue, break) : 자바와 동일
