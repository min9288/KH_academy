background-image : 요소의 배경을 이미지로 지정

선택자{background-image : url(이미지경로);}
글씨를 입력

background-repeat : 배경이미지 반복

선택자{background-repeat : repeat | repeat-x | repeat-y | no-repeat}

background-position : 배경이미지 위치조정

선택자{background-position : 수평위치 수직위치}

background-size : 배경이미지의 크기를 조절

선택자{background-size : auto | contain | cover | 크기지정}

background-attachment : 웹페이지 상하 이동 시 배경이미지 고정여부

선택자{background-attachment : scroll | fixed }

background-origin : 배경이미지 배치 기준설정

선택자{background-origin : border-box | padding-box | content-box}

background : 배경관련 속성을 한번에 설정

선택자{background : image repeat attachment position clip origin size}
