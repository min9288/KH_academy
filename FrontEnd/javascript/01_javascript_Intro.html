자바스크립트 개요

자바스크립트 작성 방식

1. inline : 자바스크립트 양이 한 줄 또는 두줄 정도로 소량일때 사용

2. internal 방식 : html파일 내 자바스크립트 소스를 작성(script태그 내부에 작성)

3. external 방식 : 자바스크립트 파일을 따로 외부 파일로 작성한 후 불러오는 방식

데이터 출력 방법

1. document.write(내용) : 브라우저 화면에 값을 출력

2. window.alert(내용) : 내용을 메세지 창에 출력(window는 생략 가능하여 alert으로 사용 가능)

3. console.log(내용) : 개발자도구 화면의 콘솔에 출력(개발 도중 값 확인을 위한 용도로 사용)

데이터 입력 방법

1. window.confirm(질문) : 질문에 대해 예/아니오의 결과를 얻을때 사용

2. window.prompt(질문) : 질문에 대해 입력할 텍스트 필드와 확인/취소 버튼이 있는 대화창 출력
