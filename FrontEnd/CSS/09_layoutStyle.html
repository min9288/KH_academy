테두리 스타일

border-style : 테두리 스타일을 지정

선택자{ border-style : none | hidden | dashed | dotted | double | groove | inset | outset | ridge | solid }

border-width : 테두리 두께

선택자{ border-[위치]-width : thin | medium | thick | 숫자 }

border-color : 테두리 색상

선택자{ border-[위치]-color : 색상값 }

border : 스타일, 두께, 색상을 한번에 지정

선택자{ border-[위치] : 두께 스타일 색상 }

border-radius : 박스 모서리를 둥글게하는 속성

선택자{ border-[위치]-radius : 숫자 }

box-shadow : box에 그림자효과를 주는 속성

선택자{ box-shadow : 수평거리 수직거리 흐림정도 번짐정도 색상 }
