작성된 html 태그 접근

엘리먼트 객체 가져오기

1. id속성 사용 : document.getElementById("아이디명")

태그의 아이디속성을 이용해서 태그 정보를 가져오며, id속성은 태그의 고유식별속성으로 동일한 아이디를 여러개 태그에 부여해도 1개 요소만 가지고 올 수 있음

2. class 속성 사용 : document.getElementsByClassName("클래스명")

클래스 속성으로 태그정보를 가져오며, 동일한 클래스 속성값을 가진 엘리먼트를 모두 가져옴 -> 배열형태
단, 해당클래스를 사용하는 속성이 1개여도 무조건 배열

3. name 속성 사용 : document.getElementsByName("name속성값")

name 속성으로 태그정보를 가져오며, 동일한 name속성값을 가진 엘리먼트를 모두 가져옴 -> 배열형태
단, 해당 name 속성을 사용하는 엘리먼트가 1개여도 무조건 배열

4. 태그 사용 : document.getElementsByTagName("태그명")

태그 이름을 이용해서 태그정보를 가져오며, 동일한 태그 엘리먼트를 모두 가져옴 -> 배열
단, 해당 태그가 1개여도 무조건 배열

엘리먼트 정보 사용

1. innerHTML 속성 : 엘리먼트 객체의 시작태그와 종료태그 사이의 내용을 문자열로 가져옴

엘리먼트 내부 내용을 읽어올 수 있고 변경도 가능

2. outerHTML 속성 : 엘리먼트 객체의 시작태그와 종료태그 사이의 내용 + 시작태그 + 종료태그 포함하여 문자열로 가져옴

엘리먼트 내용을 읽어올 수 있음 -> 수정X

3. value 속성 : 엘리먼트 객체의 value속성 값(주로 input태그에 사용)

엘리먼트의 value 값을 읽어올 수 있으며 변경 가능

엘리먼트 속성 변경

innerHTML, value 속성은 현재 값을 가져오는것 뿐만아니라 변경도 가능
단, outerHTML은 현재 값을 가져오는것만 가능하고 변경은 불가능

변경 전 메세지

엘리먼트 css 변경

CSS 속성값을 변경하는 방법 엘리먼트객체.style.속성명 = "값" 으로 적용
이때 속성명은 카멜표기법사용 (ex. background-color -> backgroundColor)
