효과

페이지에 애니메이션 효과를 만들기위한 메소드

사용방법 : $("선택자").메소드명([speed], [easing]. [callback])

speed : 실행속도(ms), easing : 변경되는 지점별 속도(linear : 일정하게 | swing : 처음과 끝이 약간 느리게), callback : 메소드 실행 후 수행할 함수

단, easing을 사용하기 위해서는 일반 jquery가 아닌 jquery-ui 라이브러리를 추가로 선언해야 함

animate() : 현재의 css속성을 설정한 값으로 차츰 변경되는 효과를 주는 메소드

$("선택자").animate({css속성}, [시간], [easing], [callback])

단, 모든 속성이 적용되는 것이 아니며 jquery-ui 라이브러리를 추가하면 더 많은 속성을 animate로 조절 가능(ex.background-color)
