CSS 선택자

선택자
특정한 HTML 태그를 선택할 때 사용하는 기능으로, 태그를 선택하여 원하는 스타일과 기능을 적용할 수 있다.

전체 선택자 (모든 선택자)

HTML 문서 안의 모든 태그를 선택할 때 사용하는 선택자

태그 선택자

HTML 문서 안의 같은 태그를 모두 선택할 때 사용

 p {color : red;} 와 같은 형태

아이디 선택자

HTML 문서 내에 해당하는 아이디 속성의 값을 가진 태그만 선택할때 사용(id는 중복X)

 #아이디명 {color : red;} 와 같은 형태
  1. 아이디 선택자 테스트1
  2. 아이디 선택자 테스트2
  3. 아이디 선택자 테스트3
  4. 아이디 선택자 테스트4
  5. 아이디 선택자 테스트5
  6. 아이디 선택자 테스트6

클래스 선택자

HTML 문서 내 여러개 태그를 동일한 클래스명으로 지정하여 선택할 때 사용

 .클래스명 {color : red;} 와 같은 형태
  • 클래스 선택자 테스트1
  • 클래스 선택자 테스트2
  • 클래스 선택자 테스트3
  • 클래스 선택자 테스트4
  • 클래스 선택자 테스트5
  • 클래스 선택자 테스트6
