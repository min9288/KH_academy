BOM(Browser Object Medel)

브라우저의 정보, URL 정보, 모니터 화면정보 등을 취득하거나 제어할 수 있는 객체

window : 자바스크립트 최상위 객체로 브자우저 차에 대한 설정을 하는 객체

window.opne : 높이와 너비를 입력하여 팝업창을 띄움

setTimeout : 일정 시간 후 함수를 한번 실행 / id값 리턴

clearTimeout(id) : setTimeout() 종료

setInterval : 일정시간마다 함수를 반복 실행 / id값 리턴

clearInterval(id) : setInterval() 종료

onload : 윈도우 객체가 로드가 완료되면 자동으로 실행되는 함수

screen 객체 : client 운영체제 화면에 대한 속성값을 가지는 객체

location 객체 : 브라우저 주소표시줄(URL)과 관련된 객체
